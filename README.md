# Deep-Dive-AI-Project
For Deep Dive AI Camp at University of Dallas Texas. 

# Soccer Sim W/Q-learning:

Basic Idea:

- Train 8 different, slightly varrianted models to control a team in soccer with Reinforcement Learning
- The Game Functions Like Fifa or Madden: as in you control player at a time, with the option to switch between players
- At the end pit them against eachother to see which is the best model, for a sort of demo

Constraints:

- Each team may have a different amount then 11, for the simple purpose of minimizing complexity
- Graphics won't be all that great, don't expect a AAA Game.
